Prueba Desarrollador(a) Full-Stack CurrencyBird

CurrencyBird quiere realizar un nuevo cotizador de moneda para sus clientes.

El cotizador está compuesto por un país de destino, el cual tiene asociada una divisa o moneda, por ejemplo en el caso de Perú la moneda asociada es el sol peruano (PEN). Además existen 2 modalidades de cotización:

1) Especificando el monto que se quiere enviar en pesos chilenos y calculando el monto que recibirá el beneficiario.
2) Especificando el monto en la moneda de destino que el beneficiario debe recibir, calculando el monto que deberá pagar el cliente.



Para realizar la cotización se debe considerar los siguientes aspectos, para cada una de las modalidades (Envías o Recibes)


Spread: Es la diferencia entre el precio de compra (venta) y el precio de venta (compra) de un par de divisas.

Margen: Es un porcentaje que se le aplicará a la cotización producto del margen de ganancia que obtendrá CurrencyBird.

Costo: Es el costo que involucra realizar una conversión de divisa.

Para simplificar el desarrollo, considerar los siguientes valores fijos y solo 3 destinos de moneda. El valor del dolar en peso fijo es de $757, con un spread dolar-peso de 0,4%.
